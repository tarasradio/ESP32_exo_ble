# ESP32 EXO BLE
Firmware for controlling an ODrive-based exoskeleton 
### Features:
 - Using [Delayed Output Feedback Control for Gait Assistance](https://www.researchgate.net/publication/333158963_Delayed_Output_Feedback_Control_for_Gait_Assistance_With_a_Robotic_Hip_Exoskeleton);
 - ODrive [MKS Mini V1.0](https://github.com/makerbase-mks/ODrive-MKS) control;
 - BLE Control;
 - Recording motion files to an SD card.
